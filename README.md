# python-cli-clickhouse-single-node-simple

## Description
Creates a small database table
called `dog` and populates.

## Tech stack
- python
  - clickhouse

## Docker stack
- python:3.11.2
- yandex/clickhouse-server:latest

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
